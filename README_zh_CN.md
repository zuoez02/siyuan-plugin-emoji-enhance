# Emoji增强插件

用于优化Emoji管理交互的插件

## 功能

+ 点击文档树中的文档图标或者文档顶部的Emoji图标，点击上传按钮，可以直接选择一个或多个文件直接导入到思源中，无需刷新
+ 支持手动操作后手动刷新
+ Emoji自定义文件夹构成的表情包组
+ 支持从URL下载图片
